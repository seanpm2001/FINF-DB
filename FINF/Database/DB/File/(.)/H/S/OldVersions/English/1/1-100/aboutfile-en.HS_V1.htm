Definition 1: Haskell Source code

A .HS file is a source code file for the Haskell programming language.

Here is an example of a Hello World program in this language:

module Main (main) where -- not needed in interpreter, is the default in a module file main :: IO () -- the compiler can infer this type definition main = putStrLn "Hello, World!"

Definition 2: Pseudo-extension (null)

A pseudo-extension is a file format that is applied to a file, despite not having a purpose, other than to change the file name (such as EDUCATION.HS or LAUG.HS) however this file extension is rarely used as a pseudo-extension
