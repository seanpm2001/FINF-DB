A .ipa file is a package file for iOS and iPhoneOS applications. Originally, with iPhone OS, there was no support for third party apps. Starting with iPhoneOS 2 (before it was called iOS) support for 3rd party apps was added through a special package format known as ipa, which stands for iOS App Store Package. Later on, support for ipa files was added to Apples MacOS operating system. As of 2021, support for ipa files is confined to iOS, iPadOS, iPhoneOS, and MacOS.
