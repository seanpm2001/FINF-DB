Definition 1: Debian package file

A .UDEB file is an Apple Disk image file. It is used to mount and install software on the MacOS operating system. It was introduced in MacOS 9.

Definition 2: Pseudo-extension (null)

A pseudo-extension is a file format that is applied to a file, despite not having a purpose, other than to change the file name (such as IAN.UDEB or BED.UDEB) however this file extension is rarely used as a pseudo-extension
